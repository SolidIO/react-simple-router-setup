# react-simple-router-setup
A simple project setup that uses react router


Setup:

1. Clone it.
2. Run "npm install".
3. Run "bower install"
4. Run "gulp serve" for livereload



To build project with minified code

1. Run "gulp clean_dist" (No demand, but nice)
2. Run "gulp"
3. All your files are in dist/
